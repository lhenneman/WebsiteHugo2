===================================================
 HAQ Lab Website — Profile Setup Instructions
===================================================

Hi! This file walks you through setting up (or updating) your profile
on the HAQ Lab website (lucashenneman.org).

You don't need to know anything about web development or Git.
Just follow the steps below and email the completed folder back to Lucas.

----
STEP 1 — Rename your folder
----
Your folder is currently named "TEMPLATE". Rename it to your GMU username
(e.g., "jsmith" for Jane Smith, j.smith@gmu.edu → use "jsmith").

Use all lowercase letters and no spaces.

----
STEP 2 — Edit your profile file
----
Open the file named "_index.md" in any plain text editor:
  - Windows: Notepad (search for it in the Start menu)
  - Mac: TextEdit (make sure it's in plain text mode: Format → Make Plain Text)
  - Or any code editor you prefer

Fill in each field between the quotes. Instructions for every field
are written as comments (lines starting with #).

IMPORTANT: Keep the --- lines at the top and bottom of the front matter exactly as they are.
IMPORTANT: For any field you're not sure about, you can leave it blank or just delete that field.

----
STEP 3 — Add your photo
----
Replace the placeholder file called "avatar.jpg" with a photo of yourself.
  - Keep the filename exactly: avatar.jpg
  - Any standard photo works; a headshot is best
  - It will be displayed as a circle, so center yourself in the frame

----
STEP 4 — Send it back
----
  1. Zip your folder (right-click → "Compress" on Mac, "Send to → Compressed folder" on Windows)
  2. Email the zip file to lucas@example.com with the subject: "Website profile — [Your Name]"

Lucas will review and publish your profile within a few days.

----
QUESTIONS?
----
Email Lucas at lhennema@gmu.edu
